# Zulu Control Board V1.1 I2C OSHW
This design was originally created to control the ZuluIDE in a 5.25" bracket. It has been relicensed under the OSHW CERN OWL-S V2 see [LICENSE](LICENSE) for the community to spin their own version or produce this design themselves.

The design is create in KiCAD and gerber, BOM, and position files can be built automatically as a Github action.
